# MCP Fundamentals

# 1. What is MCP?

**MCP (Model Context Protocol)** is an open protocol designed to connect AI models with external data sources, tools, services, and applications.

It provides a standardized way for AI systems to:
- Access resources
- Execute tools
- Retrieve information
- Manage context
- Interact with enterprise systems

MCP acts as a bridge between AI intelligence and real-world business systems.
